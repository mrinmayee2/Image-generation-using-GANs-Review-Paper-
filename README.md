## 📌 About the Project  
This repository contains a **comprehensive review paper on Generative Adversarial Networks (GANs)**, focusing on their role in **image generation**.

GANs, introduced in 2014, use two neural networks — a **Generator** and a **Discriminator** — that compete with each other to generate realistic data.

## 🔍 Topics Covered  

- Introduction to GANs  
- Adversarial learning & working mechanism  
- Training dynamics and theoretical foundations
